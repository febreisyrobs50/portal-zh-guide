# portal-zh-guide

该仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对任何特定域名或具体网站，而是作为通用指南或参考页面进行存储与分发。

## 项目简介

`portal-zh-guide` 是一个存放静态 HTML 页面的仓库。每个页面均以独立文件形式存在，可直接通过浏览器打开或部署至静态托管服务。项目旨在提供一组结构清晰、内容自包含的 HTML 资源，适用于需要快速访问或引用的场景。

## 目录说明

```
portal-zh-guide/
├── index.html          # 仓库主页（可选）
├── page-example.html   # 示例页面
├── docs/               # 相关文档或说明
└── assets/             # 静态资源（如图片、样式、脚本）
```

- `index.html`：可选的仓库入口页面，用于导航或概览。
- 其余 `.html` 文件：独立页面，各自包含完整的内容与样式。
- `docs/`：存放非 HTML 格式的辅助文档。
- `assets/`：页面引用的外部资源文件。

## 页面归档说明

- 所有 HTML 页面均为独立归档，不依赖外部数据库或后端服务。
- 页面内容以中文为主，风格简洁，适合直接阅读或部署。
- 归档范围包括但不限于：操作指南、信息汇总、参考手册等通用类型。
- 每个页面的标题和描述建议在 HTML 的 `<title>` 和 `<meta>` 标签中明确。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 对页面内容进行修正、补充或改进。
- 新增页面时，请确保文件名具有描述性，并在提交信息中简要说明页面用途。
- 如页面引用外部资源（如图片、样式表），建议将其一并放入 `assets/` 目录，避免外链失效。
- 本项目为静态资源归档性质，不涉及动态功能或用户交互。
